std::vector<std::string> common::split ( const std::string &  str)
inline

The function slices a string into substrings.

This function slices a given string into substrings and writes them to a string vector. The line feed character ("<space>") is used as a separator.

Note
If the delimiter is at the beginning of the line, then the first substring will be empty. That is, the line is empty before the separator.
Parameters
[in]str- input of data string.
Returns
Vector of substrings.
